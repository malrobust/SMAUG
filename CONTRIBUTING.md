# Contributing to Sifra

Thank you for your interest in improving Sifra! We welcome contributions from the community.

## How to Contribute

1. **Bug Reports**: If you find a bug, please open an issue describing the problem and steps to reproduce.
2. **Feature Requests**: Have an idea for a new tool or feature? Open an issue for discussion.
3. **Pull Requests**:
   - Fork the repository.
   - Create a new branch (`feature/your-feature` or `fix/your-bug`).
   - Commit your changes with clear, descriptive messages.
   - Push to your fork and submit a pull request.

## Development Setup

- Follow the installation instructions in [README.md](README.md).
- Ensure your code follows PEP 8 standards.
- Add tests for new features where possible.

## Code of Conduct

Please note that this project is released with a [Contributor Code of Conduct](CODE_OF_CONDUCT.md). By participating in this project you agree to abide by its terms.
